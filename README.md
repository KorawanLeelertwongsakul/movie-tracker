# 🎬 Movie Tracker

Movie Tracker คือเว็บแอปสำหรับบันทึกรายชื่อภาพยนตร์  
ผู้ใช้สามารถเพิ่ม แก้ไข ลบ และดูรายละเอียดของหนังได้

โปรเจกต์นี้พัฒนาโดยใช้ Flask และ Bootstrap


ผู้จัดทำ : กรวรรณ ลีเลิศวงศ์สกุล เลขนักศึกษา : 6810110005 

## ✨ Features

ระบบสามารถทำได้ดังนี้

- ➕ เพิ่มหนังใหม่
- 📝 แก้ไขข้อมูลหนัง
- ❌ ลบหนัง
- 🔍 ค้นหาหนัง
- 📄 ดูรายละเอียดหนัง
- 🖼️ อัปโหลดรูปภาพหนัง
- ⭐ ให้คะแนนหนัง
- 📊 แสดงจำนวนหนังทั้งหมด
